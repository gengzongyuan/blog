#### 等待
等待在UI自动化测试中会经常使用到，有时我们需要服务器进行响应或者js进行执行，才能执行接下来的动作。

<!--more-->


##### 1.time的固定等待`调试代码使用`
```
1、固定等待 无论是否出现，都等那么久
import time
time.sleep()
```

##### 2.隐式等待`懒人用法`
如果找不到，等待10秒，继续寻找
```
driver.implicitly_wait(10)
```

##### 3.显式等待`推荐使用`
- 基础使用
```
from selenium.webdriver.support.wait import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC
from selenium.webdriver.common.by import By
```
```
locator = (By.CSS_SELECTOR,"body")  # 定位器
method = EC.visibility_of_element_located(locator)  # 定位方法--如果定位到元素且可见
ele = WebDriverWait(driver,10,0.3).until(method)  # 设置10秒等待时间，每隔0.3秒寻找一次
print("********",ele)
```
